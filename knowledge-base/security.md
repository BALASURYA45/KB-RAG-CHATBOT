# Security

## Enable Two-Factor Authentication

To enable two-factor authentication, open Settings, select Security, and choose Two-Factor Authentication. Scan the QR code with an authenticator app and enter the verification code.

## Login Security

If you receive a login alert you do not recognize, reset your password immediately and review active sessions from Security settings.

## Session Management

You can sign out of other sessions from Settings under Security. Choose Active Sessions, review the list, and select Sign Out for any session you do not recognize.
