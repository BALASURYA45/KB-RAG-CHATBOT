# Integrations

## Connect Integration

To connect an integration, open Settings, select Integrations, choose the service, and click Connect. Follow the authorization steps shown by the external service.

Only admins can connect integrations for a shared workspace.

## Disconnect Integration

To disconnect an integration, open Settings, select Integrations, choose the connected service, and click Disconnect.

Disconnecting an integration stops future syncs but does not delete data that was already imported.

## Integration Sync Failed

If an integration sync fails, open Settings, select Integrations, and review the service status. Reconnect the integration if authorization has expired.

If sync continues to fail, create a support ticket with the integration name and the time of the failed sync.
