# Admin

## Invite Admin

To invite an admin, open Team Settings, select Invite Member, enter the email address, and choose the Admin role before sending the invite.

Admins can manage team members, security settings, and workspace configuration. Only trusted users should receive admin access.

## Change User Role

To change a user's role, open Team Settings, select Members, choose the user, and select a new role from the role menu.

Role changes take effect immediately. Users may need to refresh the application to see updated permissions.

## Locked User Account

If a user account is locked after too many failed sign-in attempts, an admin can open Team Settings, select the user, and choose Unlock Account.

The user should reset their password before trying to sign in again.

## Audit Activity

Admins can review important account events from Settings under Audit Activity. The audit log includes sign-ins, role changes, billing updates, and security changes.

Use filters to narrow events by user, date, or event type.
