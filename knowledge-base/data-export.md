# Data Export

## Export Account Data

To export account data, open Account Settings and select Data Export. Choose the data type, select the date range, and click Request Export.

Exports may take several minutes to prepare. A download link is sent to the account owner when the export is ready.

## Download Export File

When an export is ready, open Account Settings, select Data Export, and choose Download next to the prepared file.

Export links expire after 7 days. Request a new export if the link has expired.

## Delete Export File

To delete an export file, open Account Settings, select Data Export, find the file, and choose Delete Export.

Deleting an export file removes the downloadable copy but does not delete the original account data.
