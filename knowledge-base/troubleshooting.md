# Troubleshooting

## Cannot Sign In

If you cannot sign in, confirm that the email address is correct and try resetting your password. If two-factor authentication is enabled, make sure the verification code is current.

If sign-in still fails, clear browser cookies for the application, try another browser, and check whether the account has been deactivated by an admin.

## App Loads Slowly

If the app loads slowly, refresh the page and check your network connection. Close unused browser tabs and disable browser extensions that may block scripts.

If the issue continues, try an incognito window or a different browser. Report the approximate time of the issue when creating a support ticket.

## Browser Compatibility

The application works best on the latest versions of Chrome, Edge, Firefox, and Safari. Older browsers may not support all security and interface features.

If a page does not display correctly, update the browser and retry before contacting support.

## Missing Data

If expected data is missing, refresh the page and confirm that the correct workspace is selected. Some data may be hidden by filters or date ranges.

If the data is still missing, contact an account admin to confirm your role and permissions.
